# Восстановление нетлиста

Для восстановления нетлиста применяется гибридный подход.

Соединения "крупным мазками" между блоками размечаются в фотошопе по [межсоединениям](../rails.md).

Шлангам даются названия и это потом руками заносится в HDL.

Соединения внутри доменов ячеек производятся через утилиту Deroute и из неё выгружается готовый Verilog.

Для каждого домена ячеек есть соответствующий XMLZ исходник для Deroute.
